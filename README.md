# Dev
1. Clone the project

2. Copy ```.env.template``` and rename it to ```.env```

3. Run on terminal
```
yarn install
```

4. Open Docker Desktop and run the image with:
```
docker-compose up -d
```

5. Run the Nest backend with:
```
yarn start:dev
``` 

6. Visit the site at
```
localhost:3000/graphql
```
